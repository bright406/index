LESSON 1:
File and Folder - Detailed Comparison

What is a File?

A file is a named collection of data stored together. Files contain content such as text documents, images, videos or music. Files have names and usually have a file extension (for example report.docx) that tells the computer what kind of file it is and which program opens it.

What is a Folder?

A folder (also called a directory) is a container used to store and organize files and other folders (subfolders). Folders do not have file extensions — they are simply used to group related files together.

Analogy: Your computer is a school bag. Files are books and papers; folders are the pouches or subject dividers that keep each subject's books together.

Why Create Folders?

Creating folders is a simple habit that saves time and prevents mistakes. Below are clear reasons and classroom-level examples.

1. Organization

Folders keep related files together. Example: Put all your ICT assignments in ICT_Assignments so you don't mix them with other subjects.

2. Faster Searching

When files are grouped, you look in the right place instead of searching across the whole computer.

3. Avoid Accidental Loss

Well-organized folders reduce accidental deletion and duplication because you know where important files are stored.

4. Easy Backups & Sharing

You can copy or share a single folder that contains everything needed for a project — easier than selecting many files.

5. Professional Habits

Organized folders help when working in teams: each project can have its own folder, with subfolders for documents, images, and reports.

Common File Types & How They Are Used

Each file type stores different kinds of data. The extension (the letters after the dot) helps the computer decide how to open the file.

Document — .docx, .pdf, .txt — essays, reports, notes.
Spreadsheet — .xlsx, .csv — marks, tables, calculations.
Presentation — .pptx, .odp — slides for lessons.
Image — .jpg, .png, .gif — photos, diagrams.
Audio — .mp3, .wav — songs, recorded lessons.
Video — .mp4, .avi — lesson recordings, demonstrations.
System — .sys, .dll — used by the operating system.
Archive — .zip, .rar — compressed folders for transfer/backup.
Note: If you receive a file that you cannot open, check the extension and use the correct program (e.g., Word for .docx, VLC for .mp4).

Creating a Folder

To create a new folder:

  1. Go to the location where you want to create the folder (e.g., Desktop or Documents).
  2. Right-click on an empty space.
  3. From the menu, select New → Folder.
  4. Type a suitable name, for example ICT_Assignments.
  5. Press Enter to save the folder name.

This helps you organize your files neatly into subject or topic folders.

Renaming a Folder

To rename a folder:

  1. Locate the folder you want to rename.
  2. Right-click on the folder name.
  3. Select Rename from the context menu.
  4. Type the new name (e.g., change New Folder to Class6_ICT).
  5. Press Enter to confirm.

Renaming keeps folders meaningful and easier to find.

Deleting a Folder

To delete a folder:

  1. Locate the folder you no longer need.
  2. Right-click on it and choose Delete.
  3. The folder will move to the Recycle Bin.
  4. If deleted by mistake, open the Recycle Bin and click Restore.
  5. To permanently remove it, right-click the Recycle Bin and select Empty Recycle Bin.

This keeps your computer organized and free of unnecessary folders.
